| 后缀   | `() [] .`（点操作符）                     | 左到右  |
| 一元   | `expr++ expr--`                     | 从左到右 |
| 一元   | `++expr --expr + - ~ !`             | 从右到左 |
| 乘性   | `* / %`                             | 左到右  |
| 加性   | `+ -`                               | 左到右  |
| 移位   | `>> >>> <<`                         | 左到右  |
| 关系   | `> >= < <=`                         | 左到右  |
| 相等   | `== !=`                             | 左到右  |
| 按位与  | `&`                                 | 左到右  |
| 按位异或 | `^`                                 | 左到右  |
| 按位或  | \|                                  | 左到右  |
| 逻辑与  | `&&`                                | 左到右  |
| 逻辑或  | \|\|                                | 左到右  |
| 条件   | `? :`                               | 从右到左 |
| 赋值   | `= += -= *= /= %= >>= <<= &= ^=`\|= | 从右到左 |
| 逗号   | `,`                                 | 左到右  |
# 算数运算符
和`C/Cpp`规则一致
`+` `-` `*` `/` 加减乘除
`%` 取模
`++` `--` 自增符 自减符，。`a++`先用后加，`--a`先减后用
`()`使得优先计算括号内内容
## 赋值运算符
`=`  `+=` `-=` `*=` `/=`  `%=` `<<=` `>>=` `&=` `^=` `|=` 等效于`a=a+b`

---
# `boolean`运算符
`==` `!=` `>=` `>` `<` `>=` `<=`
和`C/Cpp`类似，但是该运算符会返回`true`或`flase`的`boolean`
## 逻辑运算符
`||` `&&` 前者是“或”，后者是“与"，`!`这个是非

# 条件运算符
和`C/Cpp`规则一致，使用 形如`a>b?a:b`
# 位运算符
和`C/Cpp`使用方法一致
`&` 按位与
`|`按位或
`^`按位异或
`~`按位否
`<<`按位左移
`>>`按位右移
`>>>`按位右移补零
## `instanceof`
```Java
( Object reference variable ) instanceof  (class/interface type)
```
检测左侧对象是否是右者类或者接口的一个对象
或者检测左侧对象是否是右侧类型
