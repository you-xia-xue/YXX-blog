
# 条件语句（`if`）
``` Java
if(){
}else if(){
}else {
}
```
如图，和`C/Cpp`类似，`if()`内部为布尔表达式，或为非0值就执行`if`代码块
# 选择语句`switch`
``` Java
switch(role){
	case ab: break;
	default://作为默认语句使用，如果没有该语句，case不匹配则直接跳过语句
}
```
和`C/Cpp`不同，里面只需要类型相同的常量即可
# 循环语句
和`C/Cpp`很像了
## `for`
``` Java
for(int i =0;i<5;i++)
```
# `while`

``` Java
 while(){}
```

# `do while`
``` Java
do{}while;
```
# `for each`
```java
//假设fruits是string数组
for(Sting fruit : fruits){//逐个遍历fruits值并赋值给fruit参与循环
}
```
和`C/Cpp`的`Rarge for`类似 ，缺点也类似：无法控制循环次数
# 控制语句
## `continue、break`
``` java
while(){
continue;//下一个循环
break;//中止循环
}

switch(){
case():
break;//阻止运行下一个case语句
}
```
# 捕获异常语句
```Java 
try{}//可能有异常的代码
```
