`java`中每一个文件都算是一个类（外部），我们的方法都放在文件对应的类里。我们把在外部类中建立的类叫做内部类  
`java`中的类也有抽象类（只有方法没有变量），所以是无法建立实例的。  
一个源文件有且只有一个`public`类，但可以有多个非`public`类。  
### 内部类
类内部可以再创建类，称为内部类，  
以下是**非静态内部类**:我们可以在内部类访问外部类，但是我们需要先实例化外部类，然后通过外部类创建内部类。  
``` java
public class Main{
	int a;
	public class b{
	}
	Main c = new Main();
	Main.b d = Main.new b(); 
}
```
以下是**静态内部类**：它可以直接访问外部类的静态成员，但是需要建立外部类的实例才能访问非静态成员
``` java
public class Main{
public static class TestClass{}//这是一个可以被外部类调用的静态类
}
```

## 抽象类
抽象类，它类似与`Cpp`的包含纯虚函数的类。不能被实例化，需要被继承才能被实例化，如果基类含有抽象方法，则需要重写或声明子类
为抽象类
使用`abstract`显式定义抽象类和抽象方法，且他们不能被`static`修饰
``` java
public abstract class Employee{//无法被实例化
public abstract double computePay();//必须重写，不需要花括号
}
```
