# 类型转换
## 隐式类型转换
和`C/Cpp`类似，通常由小类型转换成大类型
## 显式类型转换
和`C/Cpp`类似
``` java
float a = 4.0F;
int b = (int)a;
String c ="3344";
Integer.parseInt(c);//可以得到3344的int值
float d = 1.0f;//小数默认double类型，需要f显式说明为float
```
使用内置`java.lang`类实现字符串转换数据类型，也可以反之

# 基本类型
`byte`占1字节
`int`占4字节在java可以这样`123_456_678`
`long`占8字节
`float`占4字节
`double`占8字节
`char`占2字节，存储字符
`boolean`占1字节
``` java
byte age = 30;
long views = 3_123_456_789L;
float a = 1.0f;
char word = 'A';
boolean bool = true;
```
可以对基本类型后面加入`.`来快速生成代码
`void`无类型
# 引用类型
``` java
int a=1;
int b=a;//a,b是两块内存，只是b用了a的值
Point point1 = new Point(1,2)
Point point2 = point1;//point1和point2指向同一块内存
```
我们使用变量名引用使用`new`生成的内存，很像`Cpp`的指针。如果没有初始化，默认值是`null`

## 字符串`String`
`String`是一种类
``` Java
//这是一种类的初始化方法
String message = "Hallo world";
String message = message + "!!"
message.lenth()//调用方法以查询字符串长度
```
 字符串的方法不会影响原始字符串 
## 数组
``` Java
int[] numbers = new int[5];//new申请堆内存
int[] nunbers = {1,2,3,4,5}
Arrays.toString(numbers)
```
需要注意的是，`int[]`对应的变量存储的是内存地址，不可直接打印。需要使用`Arrays`类内的方法把相应类型转换成字符串打印.
