## 复杂类型
### 类
详细看类
``` java
//形如
public class a{
    public static void test(){
    } 
}
```
简单讲讲
1. `public class`声明一个类其中`public`说明了这个类的访问权限，后面的`a`表示这个类的名字，类中的`test()`为`a`中的一个成员函数
2. `public static void test(){}`中，`public`声明这个成员全局可见（可以直接调用），`static`说明这个成员是静态成员。

### 字符串`String`
`String`是一种类
``` Java
//这是一种类的初始化方法
String message = "Hallo world";
String message = message + "!!"  
可以使用“+”号来拼接字符串
message.lenth()//调用方法以查询字符串长度
```
 字符串的方法不会影响原始字符串 
### 数组
``` Java
int[] numbers = new int[5];//new申请堆内存
int[] nunbers = {1,2,3,4,5}
Arrays.toString(numbers)
```
需要注意的是，`int[]`对应的变量存储的是内存地址，不可直接打印。需要使用`Arrays`类内的方法把相应类型转换成字符串打印.
