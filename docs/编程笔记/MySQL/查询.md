# 查询
用形如`SELECT * FROM 表名`来查询整张表  
## 条件查询
这里可以引出基本的运算符   
`+ - x \ %` 算数运算符
重点是以下的比较运算符:  
|符号|描述|
|--|--|
|=|等于|	
|<>, !=	|不等于	
|>	|大于	
|<	|小于	
|<=	|小于等于	
|>=	|大于等于|

逻辑比较符:  
|运算符号|作用|
--|--
NOT 或 !	|逻辑非
AND	|逻辑与
OR	|逻辑或
XOR	|逻辑异或

## 多表查询
我们需要外键令表之间建立联系   
当一对一时，从表通过外键和主表建立对应关系
当多对一时，多表作为从表，通过外键和主表建立对应关系
当多对多时，通过中间表建立对应关系
