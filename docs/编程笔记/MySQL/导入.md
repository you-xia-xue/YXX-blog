以下是常用代码： 

|操作|命令|
|:--:|:--|
|创建新数据库|CREATE DATABASE 数据库名;|
|选择数据库|USE 数据库名|
|删除数据库|DROP DATABASE 数据库名;|
|建表|CREATE TABLE 表名 (列名1 数据类型 [约束], 列名2 数据类型 [约束], ...);|
|删表|DROP TABLE 表名;|
|插值|INSERT INTO 表名 (列1, 列2, ...) VALUES (值1, 值2, ...);|
|插入或替换|REPLACE INTO 表名(列1, 列2, ...) VALUES (值1, 值2, ...);|
|插入或更新|INSERT INTO students (列1, 列2, ...) VALUES (值1, 值2, ...) ON DUPLICATE KEY UPDATE 列1=值1, 列2=值2, ...;|
|更新数据|UPDATE 表名 SET 列1 = 值1, 列2 = 值2, ... WHERE 条件;|
|删值|DELETE FROM 表名 WHERE 条件;|
|查整表|SELECT * FROM 表名|
|查值|SELECT 列1, 列2, ... FROM 表名 WHERE 条件;|

建表我们可以举例子(来自菜鸟教程)  
```
CREATE TABLE 用户（
id INT AUTO_INCREMENT PRIMARY KEY，
用户名 VARCHAR（50） NOT NULL，
电子邮件 VARCHAR（100） NOT NULL，
出生日期 DATE，
is_active 布尔默认值 TRUE
);
```
该例子创建了一个表，逐行讲解疑惑词汇：  
`INT`说明了id的类型  
`AUTO_INCREMENT`说明了id的类型自增  
`PRIMARY KEY`说明id为主键（确保值唯一）  
`VARCHAR(50)`说明用户名为变长字符串（类型）  
`NOT NULL`声明该类型不能为空·
